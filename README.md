# Two Quill Stories

Professional author website for Machugari Yashwanth Reddy.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- React Icons

## Routes

- `/`
- `/books/she-was-the-friend-i-dreamed-for`
- `/books/what-love-reveals`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
